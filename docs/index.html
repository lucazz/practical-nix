Practical Nix¶

This is a practical guide on how to build, package, and deploy software using Nix.

If you don’t know what Nix is or why you should use it, read Introduction first. Getting Started will get you through installation and basic usage of Nix. Anatomay of a Derivation shows what a package specification in Nix looks like.

Once you are done with the basics, jump over to the build environment that is relevant to you, and pick an appropriate deployment target. That should cover the overall process of building and shipping a software package using Nix.
